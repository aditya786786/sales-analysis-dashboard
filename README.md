## 🧠 Project Overview

**Objective**: To analyze and visualize the sales data to track performance metrics such as sales, profit, quantity, discount, customer behavior, regional trends, and shipping efficiency.

**Tools Used**:
- 🧮 Power BI
- 📊 Excel
- 📁 DAX & Power Query
- 🎨 Custom JSON Themes
